# Chat

## О сервисе
Здесь находятся сервисы для работы с чатом и сообщениями

## Сервисы внутри
### addMessage
Добавляет сообщения в коллекцию сообщений
### createChat
Добавляет чат в коллекцию чатов
### getChats
Возвращает массив чатов для авторизованного пользователя в формате JSON
### getMessages
Возвращает массив сообщений для конкретного чата в формате JSON. Запрос обрабатывает параметры запроса <code>chatId</code> и <code>amount</code><br>
<code>amount</code> - ограничение количество сообщений до этого количества
### updateChat
Обновляет чат
### websocket
Позволяет открыть websocket-соединение. Посылает сообщения, в зависимости от события, на другие сервисы (addMessage, createChat, updateChat)

### models
